The goal of these projects is to build practical experience in programming, algorithmic thinking, and analytical problem solving.

### 💻C Programming

Projects focused on core computer science concepts such as algorithms, program logic, and low-level programming.

Projects include:

**➗Prime Divisor Algorithm**  
Implementation of an algorithm that finds the largest prime divisor of a number without using division or modulo operations.

**🕹️Pong Game**  
Terminal-based implementation of the classic Pong game demonstrating program structure, loops, and game state logic.

**🔍Cipher Program**  
Implementation of classical text encryption techniques using command-line input and character manipulation.

**🗳️Runoff Election Simulation**  
Simulation of a ranked-choice voting system that determines a winner through iterative vote redistribution.

---

### 📊SQL Data Analytics

Projects focused on analytical queries used in business and product analytics.

**ABC Analysis**  
SQL implementation of product segmentation based on revenue contribution using the Pareto principle.

**N-Day Retention Analysis**  
Query-based calculation of user retention over a defined number of days to measure engagement.

**Rolling Retention Analysis**  
Retention analysis measuring whether users return to a product after a given period.
